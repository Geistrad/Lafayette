Architecture

The Lafayette Server is a JEE web application tested on Apache Tomcat 7. The web application itself is based on the Jersey sevlet container for seving the REST API. Additionally there are some JSPs and static web resources.

Vision

There will be:

  • Web Front Ends
    • a home site (mostly static)
    • one for registration
    • one for administration
  • REST Service
    • service discovery (to find a service)
    • services
      • data
      • file
  • Domain Model
    • Data Mapper to SQL [internal]
    • Business Model (Entities: User, Role) [exported]
    • Business Service (uses data mapper) [exported]
  • Web Components
    • Thins related to HTTP.
  • Core Components
    • Things shared by all parts of the server.
